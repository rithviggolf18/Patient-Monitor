The GtkFontChooserDialog widget is a dialog for selecting a font.

GtkFontChooserDialog implements the GtkFontChooser interface and does not provide much API of its own.

To create a GtkFontChooserDialog, use gtk_font_chooser_dialog_new().

GtkFontChooserDialog as GtkBuildable

The GtkFontChooserDialog implementation of the GtkBuildable interface exposes the buttons with the names “select_button” and “cancel_button”.
