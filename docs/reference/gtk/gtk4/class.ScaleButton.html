Class

GtkScaleButton

Description [src] 

class Gtk.ScaleButton : Gtk.Widget {
  parent_instance: GtkWidget
}

GtkScaleButton provides a button which pops up a scale widget.

This kind of widget is commonly used for volume controls in multimedia applications, and GTK provides a GtkVolumeButton subclass that is tailored for this use case.

CSS nodes

GtkScaleButton has a single CSS node with name button. To differentiate it from a plain GtkButton, it gets the .scale style class.
